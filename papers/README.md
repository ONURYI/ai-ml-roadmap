# Papers

All papers read, with personal summaries and critical notes.

**Reading method:** Three passes per paper (Keshav 2007).
1. Title, Abstract, Introduction, Conclusion, Figures — 10 min
2. Full read, mark unclear sections — 1h
3. Derive every formula, challenge every claim — deep

## Paper Template

Every paper gets its own file with this structure:

```
# Paper Title

**Authors:**
**Year:**
**Link:**
**Status:**

## The One New Idea
(in one sentence)

## Problem it solves
(what was the limitation before?)

## Method
(how does it work — in my own words)

## Key Formulas
(derived myself, with explanation)

## Results
(what do the numbers show?)

## Limitations
(what did they NOT test? what could be improved?)

## My Ideas
(what would I do differently?)

## Connections
(which other papers does this build on / connect to?)
```
